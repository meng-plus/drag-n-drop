简易的移植说明
1.将 drag-n-drop 文件夹复制到工程的根目录
2. 将下列清单添加到工程
```bash
error.c
file_stream.c
flash_decoder.c
flash_manager.c
flash_intf.c
vfs_manager.c
vfs_user.c
virtual_fs.c
intelhex.c
```
2. 初始化文件系统
```c
vfs_mngr_init(true);
```
3. 构建文件系统
```c
vfs_user_build_filesystem();
```
4. 处理文件变更
```c
vfs_user_file_change_handler(const vfs_filename_t filename, vfs_file_change_t change, vfs_file_t file, vfs_file_t new_file_data);
```
5.接入msc功能-获得容量
```c
#define BLOCK_SIZE 512
*block_size = BLOCK_SIZE;
*block_num = vfs_get_total_size() / *block_size
```
6. 接入msc功能-读取数据
```c
vfs_read(sector, buf, num_of_sectors);
```
7. 接入msc功能-写入数据
```c
vfs_write(sector, buf, num_of_sectors);
```
